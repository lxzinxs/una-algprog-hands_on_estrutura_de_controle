# 💻 Exercícios de Lógica de Programação - Portugol

Este repositório contém a resolução de 10 exercícios práticos desenvolvidos para a disciplina de Lógica de Programação. Os algoritmos foram criados utilizando o **Portugol Studio** e abrangem desde conceitos básicos até estruturas de repetição e condicionais avançadas.

## 🛠️ Tecnologias Utilizadas

* **Linguagem:** Portugol (Pseudocódigo)
* **Ambiente de Desenvolvimento:** [Portugol Studio](http://univali.br/portugolstudio)
* **Conceitos Aplicados:**
    * Estruturas Condicionais (`se`, `senao`)
    * Laços de Repetição (`para`, `enquanto`)
    * Operadores Matemáticos e de Módulo (`%`)
    * Validação de Dados e Menus Interativos

## 📋 Lista de Exercícios

| Arquivo | Descrição | Destaque Técnico |
| :--- | :--- | :--- |
| `Exercicio_1.por` | Identifica se um número é Par ou Ímpar. | Uso do operador de resto `% 2`. |
| `Exercicio_2.por` | Compara dois números e exibe o maior deles. | Estrutura de decisão simples. |
| `Exercicio_3.por` | Calculadora funcional com menu de operações. | Tratamento de erro para divisão por zero. |
| `Exercicio_4.por` | Validação de nota para garantir que esteja entre 0 e 100. | Filtro de entrada com operadores lógicos. |
| `Exercicio_5.por` | Leitura de números até que o dígito 0 seja inserido. | Implementação de laço com sentinela. |
| `Exercicio_6.por` | Simulador de Caixa Eletrônico (Depósito, Saque e Saldo). | Verificação de saldo e bloqueio de valores negativos. |
| `Exercicio_7.por` | Cálculo de desconto baseado em categorias de cliente. | Diferenciação entre Comum, Premium e Funcionário. |
| `Exercicio_8.por` | Contagem de números positivos, negativos e zeros em 10 entradas. | Uso do laço `para` com contador fixo. |
| `Exercicio_9.por` | Soma e exibição de números em sequência de 1 até N. | Repetição dinâmica baseada na entrada do usuário. |
| `Exercicio_10.por` | Contagem regressiva de N até 1 com exibição de valores. | Uso de decremento no laço de repetição. |

## 🚀 Como Executar

1.  Faça o download e instale o **Portugol Studio**.
2.  Baixe os arquivos `.por` deste repositório.
3.  Abra o arquivo desejado no Portugol Studio através do menu `Arquivo > Abrir`.
4.  Pressione a tecla **F9** ou clique no ícone de "Play" para iniciar a execução.
